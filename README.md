# MealMatch 🥗

**MealMatch** is a desktop application designed to help users find the perfect recipe based on the ingredients they have on hand.

---

## 🎯 Project Overview
* **Mission:** Simplify home cooking by allowing users to find recipes based on the ingredients they already have.
* **Core Concept:** A JavaFX desktop app with a SQL database that filters recipes by dietary needs (Vegan, Vegetarian, Breakfast, snacks, etc.) and allows personal recipe management.
* **Target:** Students and busy people looking for quick and dietary-specific meals.

## ✨ Key Features
* **Smart Search:** Find recipes by title or by entering a list of available ingredients.
* **Advanced Filtering:** Toggle between categories such as Vegan, Vegetarian, Breakfast, Snacks, and more.
* **User Management:** Create personal profiles to save "Favorite" recipes.
* **Recipe Creator:** Users can contribute to the database by uploading their own custom recipes.

## 💻 Tech Stack
* **Language:** Java
* **GUI Framework:** JavaFX (utilizing Scenes, Stages, and Groups)
* **Database:** SQL (for persistent storage of recipes and user data)
* **Version Control:** Git & GitHub
